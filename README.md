基本编辑
移动光标：方向键 / Ctrl+←/→（按词移动）/ Home, End, PageUp, PageDown
选中：Shift+方向键 / Ctrl+Shift+←/→（按词选中）
复制/剪切/粘贴：Ctrl+C / Ctrl+X / Ctrl+V
撤销/重做：Ctrl+Z / Ctrl+Y
多光标与多行编辑
添加光标：Alt+点击 / Ctrl+Alt+↓ 或 ↑
选中所有匹配：Ctrl+F2
选中下一个匹配：Ctrl+D
查找与跳转
查找：Ctrl+F
替换：Ctrl+H
跳转到行：Ctrl+G
跳转到文件：Ctrl+P
跳转到符号：Ctrl+Shift+O
跳转到定义：F12
返回上一个位置：Ctrl+-
代码操作
自动格式化：Shift+Alt+F
注释/取消注释：Ctrl+/
折叠/展开代码：Ctrl+Shift+[ / ]
代码补全：Ctrl+Space
窗口与标签页管理
新建文件：Ctrl+N
关闭标签页：Ctrl+W
下一个/上一个标签页：Ctrl+Tab / Ctrl+Shift+Tab
拆分编辑器：Ctrl+\
切换编辑器焦点：Ctrl+1 / Ctrl+2 / Ctrl+3
命令面板
打开命令面板：Ctrl+Shift+P
